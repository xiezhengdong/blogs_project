Javascript常用的技巧

2017/6/15 14:57:13 34 梦影雾花

1.使用+将字符串转换成数字

这个技巧非常有用，不过这个字符串里面必须是数字，否则将返回NaN。

function toNumber(strNumber) {  
    return +strNumber;  
}  
console.log(toNumber("123"));    //输出 "123"  
console.log(toNumber(+"123"));   //输出 123  
console.log(toNumber(+"a"));     //输出 NaN

这个也适用于Date,它将返回时间戳数字。

console.log(+new Date());    //输出 1462775218918

2.在循环中缓存array.length

循环一个数组，大家一般都会这样写。

for(var i = 0; i < array.length; i++) {  
    console.log(array[i]);  
}

但是这样做数组在每次迭代的时候，都会重新计算数组的大小，这将会导致一些延误。为了避免这种现象出现，一般会把array.length做一个缓存，也就是放到一个变量里面。

var length = array.length;  
for(var i = 0; i < length; i++) {  
    console.log(array[i]);  
}

也可以这样写。

for(var i = 0, length = array.length; i < length; i++) {  
    console.log(array[i]);  
}
